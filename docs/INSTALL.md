# ChronoMate 2026 Installation Guide

## Introduction

Thank you for downloading ChronoMate 2026.

ChronoMate is a lightweight, offline chronograph recording application designed specifically for air rifle shooters. It records shot strings, calculates muzzle energy, stores rifle and ammunition information, and generates professional printable reports.

ChronoMate has been designed to be:

* Lightweight
* Fast
* Offline
* Easy to use
* Easy to maintain

No installation is required.

---

## System Requirements

ChronoMate has been tested using:

* Windows 10
* Windows 11

Supported desktop browsers:

* Mozilla Firefox
* Microsoft Edge
* Brave Browser

JavaScript must be enabled.

---

## Installation

1. Download the latest ChronoMate release from GitHub.

2. Extract the ZIP archive to any folder.

3. Keep the folder structure intact.

Example:

```text
ChronoMate/
│
├── ChronoMate.html
├── app.js
├── report.js
├── session.js
├── style.css
├── README.md
├── LICENSE
├── assets/
├── data/
│   └── ammo.js
└── docs/
    ├── INSTALL.md
    ├── USERGUIDE.md
    ├── BACKUP.md
    ├── CHANGELOG.md
    └── FAQ.md
```

4. Double-click **ChronoMate.html**.

No installation is performed.

No files are copied into Windows.

No registry entries are created.

---

## Updating

Before installing a newer version:

1. Open ChronoMate.
2. Select **Export Data**.
3. Save your backup file.

Replace the application files with the new release.

Open ChronoMate.

Select **Import Data** to restore your saved data.

---

## Supported Platforms

ChronoMate is intended for desktop operating systems using modern desktop browsers.

Current supported environments:

* Windows

Other desktop operating systems may also work but have not yet been fully tested.

---

## Mobile Devices

ChronoMate is not currently supported on Android or iOS when launched directly from local storage.

Mobile operating systems often restrict local HTML applications from accessing linked JavaScript, CSS and image files.
